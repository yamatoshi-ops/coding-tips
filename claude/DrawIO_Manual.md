# DrawIO 使用マニュアル

## DrawIOとは

DrawIO（現在はdiagrams.netとして知られている）は、無料で使えるオープンソースの図表作成ツールです。ブラウザ上で動作し、インストール不要で様々な種類のダイアグラムを作成できます。

## 主な特徴

- **完全無料**: 商用利用も含めて無料で使用可能
- **オンライン/オフライン対応**: ブラウザ版とデスクトップアプリ版が利用可能
- **クラウド保存対応**: Google Drive、OneDrive、Dropboxなどに保存可能
- **多様なフォーマット**: PNG、JPEG、SVG、PDF、XMLなど多数の形式でエクスポート可能
- **豊富なテンプレート**: 様々な図表テンプレートが用意されている

## 基本的な使い方

### 1. アクセス方法

```
https://app.diagrams.net/
```

または

```
https://draw.io/
```

### 2. 新規ファイルの作成

1. サイトにアクセス
2. 保存先を選択（デバイス、Google Drive、OneDriveなど）
3. テンプレートを選択、または空白から開始

### 3. 図形の追加

- 左側のパネルから図形をドラッグ&ドロップ
- 検索機能で目的の図形を探す
- カスタム図形のインポートも可能

### 4. 接続線の追加

- 図形をホバーすると表示される矢印アイコンをクリック
- 別の図形にドラッグして接続
- 接続線のスタイルは右側のパネルで変更可能

### 5. テキストの編集

- 図形をダブルクリックでテキスト編集モード
- フォント、サイズ、色などを変更可能

## よく使われる図形ライブラリ

- **General**: 基本的な図形（四角、円、矢印など）
- **Flowchart**: フローチャート用の図形
- **UML**: UML図作成用の図形
- **Network**: ネットワーク機器のアイコン
- **AWS**: AWSサービスのアイコン
- **Azure**: Azureサービスのアイコン
- **GCP**: Google Cloudのアイコン
- **Kubernetes**: K8sのアイコン

## エクスポート形式

- **PNG**: 画像として保存（プレゼンや文書に埋め込み）
- **JPEG**: 画像として保存
- **SVG**: ベクター形式（拡大しても劣化しない）
- **PDF**: 印刷や共有に適した形式
- **XML**: DrawIOの編集可能な形式
- **HTML**: Webページとして公開

## Tips & Tricks

### 1. グリッドとスナップ
- グリッドを表示して整列しやすくする
- オブジェクトを自動的にグリッドに合わせる

### 2. レイヤー機能
- 複雑な図は複数のレイヤーに分けて管理
- レイヤーの表示/非表示で見やすさを向上

### 3. テンプレートの活用
- 既存のテンプレートをカスタマイズして時間短縮
- よく使う図形はカスタムライブラリとして保存

### 4. バージョン管理
- Google Driveなどと連携してバージョン履歴を管理
- チーム内での共同編集も可能

### 5. 埋め込みとリンク
- 作成した図をWebページに埋め込み可能
- 図の中にハイパーリンクを設定可能

## 実践例

### システム構成図の作成例

1. AWSライブラリを開く
2. VPC、EC2、RDS、S3などのアイコンを配置
3. 接続線で関係性を示す
4. セキュリティグループやサブネットを色分け
5. 説明テキストを追加
6. PNG形式でエクスポートしてドキュメントに追加

### フローチャートの作成例

1. Flowchartライブラリを開く
2. 開始/終了、処理、判断などの図形を配置
3. 矢印で処理の流れを接続
4. 各ステップに説明を記入
5. 条件分岐にはYes/Noのラベルを追加
6. PDFでエクスポートしてレビュー資料として共有

## まとめ

DrawIOは、エンジニアからビジネスパーソンまで幅広く使える図表作成ツールです。無料でありながら機能が豊富で、システム設計からプレゼンテーション資料まで様々な用途に活用できます。クラウド保存にも対応しているため、チーム内での共有や共同編集も容易です。

ぜひ日々の業務や学習に活用してみてください。
